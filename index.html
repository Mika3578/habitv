Habitv Maven Repository Mirror

This is a Maven repository mirror for Habitv project.

Repository Information

  • Original repository: http://dabiboo.free.fr/repository/com/dabi/habitv/
  • Mirror repository: https://mika3578.github.io/habitv/repository/com/dabi/habitv/

Maven Configuration

Repository Configuration

<repositories>
    <repository>
        <id>habitv-github</id>
        <name>Habitv Maven Repository on GitHub</name>
        <url>https://mika3578.github.io/habitv/repository/</url>
    </repository>
</repositories>

Dependency Example

<dependency>
    <groupId>com.dabi</groupId>
    <artifactId>habitv-api</artifactId>
    <version>4.1.0</version>
</dependency>
